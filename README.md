# 🩸 Digital Blood Bank

A full stack web application that helps connect blood donors with patients and hospitals during emergencies.

## Features

* Donor registration
* Search donors by blood group
* Blood request system
* Admin management
* Secure backend APIs

## Tech Stack

Frontend: React.js
Backend: Node.js + Express.js
Database: MongoDB

## Installation

Clone the repository

git clone https://github.com/TEJAS-PARKAR/digital-blood-bank.git

Install backend dependencies

cd server
npm install
npm run dev
